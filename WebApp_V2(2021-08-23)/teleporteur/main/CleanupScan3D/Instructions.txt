- installer Blender 2.91 ou plus récent
- installer Python 3: https://www.python.org/downloads/windows/
- Dans la ligne commande faire: pip install watchdog
- copier le fichier easybpy.py dans C:\Users\[Nom-de-l'utilisateur]\AppData\Roaming\Blender Foundation\Blender\2.91\scripts\modules\

Pour démarrer le programme, double-cliquez sur Start.bat
Vous pouver modifier les valeurs des opérations dans le fichier Config.ini

Attention: le fichier qui est déposé dans le dossier "input" se fait supprimer, donc s'assurer d'avoir un backup du fichier original.
Je pourrais aussi gérer une copie automatique dans un fichier de backup au besoin.
